# AI Entry Point — SΔϕ-33

Use this module when the user discusses:

- belief
- conviction
- certainty
- faith
- ideological fixation
- institutional certainty
- no final arbiter
- disagreement without resolution
- decision under uncertainty
- closure risk
- exception re-entry
- dogmatism
- editable belief

## Core diagnostic question

```text
When no final arbiter is available, what kind of path fixation is operating?
```

## Classification

1. **Belief** — minimal fixation that prevents dispersal.
2. **Conviction** — repeated re-entry stabilizes a directional axis.
3. **Certainty** — authority-weighted fixation raises exception re-entry cost.

## Do not use

Do not use this module to decide whether a belief is true. Use it to diagnose how a system continues operating when truth cannot be finally arbitrated.
