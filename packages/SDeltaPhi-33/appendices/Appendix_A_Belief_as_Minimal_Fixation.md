# Appendix A — Belief as Minimal Fixation

Belief is used here as the minimal fixation point required when operation would otherwise disperse under absent arbitration. This appendix treats belief as operation preservation, not truth possession.

## Illustrative uses

- A researcher uses a provisional hypothesis when evidence is incomplete.
- A policymaker selects an initial path when full consensus is unavailable.
- An AI agent holds a temporary assumption when external verification is not currently possible.

## Diagnostic caution

Do not infer that belief is irrational merely because it is fixation. The relevant question is whether the fixation remains editable when exceptions re-enter.
