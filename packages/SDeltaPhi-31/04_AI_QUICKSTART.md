# AI Quickstart

Use SΔϕ-31 when a closed default has already been diagnosed and the task asks whether it has actually been reopened.

## Primary route

```text
SDeltaPhi-30 -> SDeltaPhi-31
```

## Supporting route

```text
SDeltaPhi-56 -> cost assumption
SDeltaPhi-44 -> path reopening vs symbolic disruption
SDeltaPhi-68 -> hidden reopening cost
```

## Ask first

```text
Was the anomaly recognized as trigger?
Who assumed the cost?
Did an alternative path become operational?
Does re-entry persist beyond the first rupture?
```

## Do not ask only

```text
Was the system criticized?
Was a petition submitted?
Was an exception granted once?
```

Those may indicate symbolic rupture, not reopening.
