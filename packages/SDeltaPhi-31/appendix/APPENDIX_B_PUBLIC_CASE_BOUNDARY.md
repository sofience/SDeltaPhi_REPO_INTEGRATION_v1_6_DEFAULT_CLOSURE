# Appendix B — Public Case Boundary

The public petition case must not be used as:

```text
a judgment that the petition should have been accepted
a judgment that the petition should have been rejected
a gender-policy argument
a party-political judgment
a claim that the government acted illegally
a claim that the issue was fully ignored
```

The case is used only as a structural example of SΔϕ-31:

```text
visibility without operational reopening
```

Before any formal historical use, factual claims should be rechecked against primary or contemporaneous sources.
