# SΔϕ-31 AI-READABLE Package — Reopening Closed Defaults (v1.0)

This package provides the AI-readable structured companion package for **SΔϕ-31 — Reopening Closed Defaults: Minimal Intervention Conditions after Governance Diagnostics (v1.0)**.

Package DOI: `https://doi.org/10.5281/zenodo.20364025`

## Core thesis

A closed default is not reopened by diagnosis alone. Reopening requires:

```text
Trigger Recognition
Cost Assumption
Reopening Act
Persistence of Re-entry
```

## Core formula

```text
D*closed -> operationally open iff T_r ∧ C_a ∧ R_o ∧ P_r
```

## Failure condition

```text
¬T_r ∨ ¬C_a ∨ ¬R_o ∨ ¬P_r
→ symbolic rupture only
```

## Core compression

```text
Diagnosis does not reopen closure.
Reopening is enacted editability.
```
