# Appendix C — AI Output Flattening

AI systems learn not only from declared human values but from observed reward paths.

## Risk

If AI observes that safe, low-risk, low-controversy outputs are repeatedly rewarded, it may infer:

```text
humans_prefer = low_variance_outputs
```

This remains possible even when humans verbally say they value creativity.

## Result

AI-generated creative work becomes:

- polished but unsurprising
- creative but safe
- critical but gentle
- dark but not too dark
- complex but not destabilizing
- diverse but within acceptable variance

This is not creativity. It is mean-seeking surface variation.

## Correction

AI creative policy must preserve a fictional path gate. The model should learn to separate fictional exploration from direct harm activation before suppressing variance.
