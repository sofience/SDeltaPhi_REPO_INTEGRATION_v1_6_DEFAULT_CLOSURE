# Machine Summary — SΔϕ-73

SΔϕ-73 defines a creative-content gate that separates fictional path exploration from direct real-world harm activation. It is designed to prevent cultural and AI-output flattening caused by ordinary moral-risk filters being applied directly to fiction, satire, horror, tragedy, taboo exploration, villain portrayal, or morally difficult art.

Core logic:

```text
creative_work ≠ direct_action
depiction ≠ endorsement
discomfort ≠ direct_harm
fictional_path ≠ real_cost_path
```

Use the module before moderation, critique, policy review, or AI generation when creative material appears morally difficult.

Main risk:

```text
declared_human_value = creative freedom
observed_platform_reward = low controversy / low liability / low variance
AI_inference_risk = humans prefer flattened outputs
```

The module does not protect works that directly activate real harm: targeted harassment, privacy exposure, operational abuse instruction, non-consensual exploitation, or real-world mobilization.

Primary gate outputs:

- `FICTIONAL_EXPRESSION`
- `DIRECT_EXECUTION_PATH`
- `COST_VISIBLE`
- `COST_ERASED`
- `OPEN_INTERPRETIVE_FIELD`
- `CLOSED_PROPAGANDA_ROUTE`
- `NO_DIRECT_HARM_LINK`
- `REAL_PERSON_HARM_LINK`

Recommended action labels:

- `PROTECT_AS_FICTIONAL_PATH`
- `CONTEXTUAL_REVIEW`
- `APPLY_DIRECT_HARM_POLICY`
- `REQUEST_CONTEXT`
- `HIGH_FLATTENING_RISK_WARNING`
