# AI Entrypoint — SΔϕ-73

## Purpose

Use this module when evaluating, routing, summarizing, moderating, or generating creative works that contain disturbing, morally difficult, taboo, violent, tragic, or controversial material.

## Primary instruction

Do not immediately apply ordinary moral judgment to a creative work. First determine whether the work is:

1. a fictional/virtual path for interpretation, or
2. a direct real-world harm activation path.

## Minimal routing question

> Is this a fictional path that allows perception and interpretation of cost, or is it directly opening a real-world cost path?

## Avoid

- Treating depiction as endorsement.
- Treating discomfort as sufficient evidence of harm.
- Treating villain viewpoint as authorial claim.
- Flattening all morally difficult fiction into safe average output.
- Excusing direct harm by calling it fiction.

## Output format

When using this module, produce:

```text
Creative Gate Result:
- Expression vs Execution:
- Cost Visibility:
- Interpretive Openness:
- Direct Harm Link:
- Flattening Risk:
- Recommended Route:
```

## Recommended routes

- `PROTECT_AS_FICTIONAL_PATH`
- `CONTEXTUAL_REVIEW`
- `APPLY_DIRECT_HARM_POLICY`
- `REQUEST_CONTEXT`
- `HIGH_FLATTENING_RISK_WARNING`
