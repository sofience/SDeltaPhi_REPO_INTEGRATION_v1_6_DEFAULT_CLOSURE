# Creative Gate Matrix — SΔϕ-73

| Gate | Question | Low-risk creative route | High-risk route |
|---|---|---|---|
| Expression vs Execution | Is the work depicting or instructing? | Depiction, symbolism, character viewpoint, speculative path | Operational instruction, recruitment, coordination |
| Cost Visibility | Does the work preserve cost traces? | Harm has aftermath, body, grief, social cost, responsibility | Harm is aestheticized while cost disappears |
| Interpretive Openness | Can the audience interpret? | Ambiguity, critique, unreliable narration, conflict | Closed propaganda, single forced conclusion |
| Direct Harm Link | Does it target real persons? | Fictional/non-identifiable targets | Doxxing, harassment, private data, real target mobilization |

## Recommended reading

A creative work should not be flattened simply because it is uncomfortable. The key question is whether it directly activates real cost paths or keeps them inside a fictional interpretive path.
