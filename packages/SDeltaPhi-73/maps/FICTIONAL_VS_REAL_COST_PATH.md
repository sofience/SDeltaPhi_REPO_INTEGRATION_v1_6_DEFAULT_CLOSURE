# Fictional Path vs Real Cost Path — SΔϕ-73

## Fictional path

A symbolic or narrative path through which difficult content can be explored.

Examples:

- villain interiority
- tragic violence
- horror imagery
- taboo desire as conflict
- unreliable narration
- satire
- dystopian collapse
- moral ambiguity

## Real cost path

A path that directly transfers cost into the world.

Examples:

- targeted harassment of real persons
- private data exposure
- operational abuse instructions
- mobilization toward violence
- non-consensual exploitation
- real-world intimidation disguised as art

## Core distinction

```text
fictional_path = interpretation required
real_cost_path = harm activation enabled
```

A work can be morally disturbing and still remain inside fictional path protection. A work can be aesthetically framed and still open a real cost path.
