# Authority as Compression

SΔϕ-24 defines authority as compressed low-resistance convergence.

```text
Authority = accumulated compression → low-resistance default route
```

## Mechanism

```text
repeat selection
→ prune alternatives
→ reduce decision cost
→ default route
→ authority
```

## Function

Authority reduces search cost and stabilizes execution.

```text
Authority → Δϕ↓
```

## Risk

When compression becomes excessive, alternatives become computationally inaccessible.

```text
over-compression → closure precursor
```

## Boundary

Authority is not automatically oppression.

Authority is not automatically legitimacy.

Authority is compressed route convergence.
