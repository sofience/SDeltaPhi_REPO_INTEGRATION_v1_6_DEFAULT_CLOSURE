## Role

SΔϕ-24 is the **authority-innovation duality / compression-reopening / misfit trigger layer** of the Sofience-DeltaPhi Formalism.

It formalizes authority and innovation as dual regimes of compression and re-opening.

## Core compression

```text
Authority = compressed low-resistance convergence. Δϕ↓

Innovation = misfit-triggered re-opening. Δϕ↑

Conflict = phase signal, not moral dispute.

Authority forms through accumulated compression.

Innovation appears when accumulated misfit forces previously pruned routes to become computable again.
```

## Core warning

```text
Authority is not automatically bad.

Innovation is not automatically good.

Conflict is not moral dispute by default.

Conflict is a phase signal that may activate re-opening.
```
