# Minimal Definitions

## Auth — Authority

A low-resistance default route produced by accumulated compression.

```text
Auth = compressed attractor route
```

## C — Compression

Route-space reduction that lowers Δϕ variability and execution cost.

```text
C → Δϕ↓
```

## Inv — Innovation

A re-opening event where alternatives become computable again.

```text
Inv = re-opening event
```

## O — Re-opening

Transition that restores computability of previously pruned routes.

```text
O = restore computability of pruned routes
```

## ε — Misfit

Mismatch signal between model and environment; accumulated coherence failure.

```text
ε = model-environment mismatch / coherence failure
```

## τ — Trigger

Threshold event activating re-opening.

```text
τ ∈ {conflict, error, non-resonance, external perturbation}
```

## Dᴀ — Authority Debt

Revision resistance accumulated by rigid authority.

```text
Dᴀ↑ → edit cost↑
```

## CE Amplification

In coupled environments, ε, τ, and Dᴀ propagate and amplify across systems.
