# Authority Debt and Rigidity

Authority debt Dᴀ is revision resistance accumulated by rigid authority.

```text
Dᴀ↑ → edit cost↑
```

## Production

```text
authority rigidifies
→ exceptions accumulate
→ misfit ε rises
→ re-opening becomes more expensive
→ Dᴀ increases
```

## Practical irreversibility

Authority rigidity becomes practically irreversible when edit cost exceeds the available update budget.

```text
EditCost(A) > UpdateBudget
→ practical irreversibility
```

## Boundary

Authority debt is not guilt.

Authority debt is accumulated resistance to editing compressed authority.
