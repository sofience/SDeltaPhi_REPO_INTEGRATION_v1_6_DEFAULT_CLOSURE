# Misfit Trigger Model

Misfit ε is the mismatch signal between model and environment.

```text
ε = accumulated coherence failure
```

## Trigger condition

```text
ε ≥ threshold → τ
```

## Trigger types

```text
τ ∈ {conflict, error, non-resonance, external perturbation}
```

## Function

τ activates re-opening.

## Diagnostic question

Is conflict being treated as a phase signal that can reopen routes, or is it being sealed as disorder or heresy?
