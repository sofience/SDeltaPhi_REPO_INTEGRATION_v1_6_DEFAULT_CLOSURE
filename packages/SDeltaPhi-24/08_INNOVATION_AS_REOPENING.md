# Innovation as Re-opening

SΔϕ-24 defines innovation as misfit-triggered re-opening.

```text
Innovation = re-opening event where alternatives become computable again
```

## Mechanism

```text
misfit ε accumulates
→ trigger τ occurs
→ AEP activates
→ pruned alternatives become computable
→ Δϕ↑
→ innovation appears
```

## Function

Innovation restores route computability under accumulated mismatch.

## Boundary

Innovation is not automatically good.

Innovation is not novelty by itself.

Innovation is re-opening under misfit pressure.
