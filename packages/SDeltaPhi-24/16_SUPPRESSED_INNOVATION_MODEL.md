# Suppressed Innovation Model

Suppressed innovation occurs when a re-opening trigger is sealed as illegitimate.

```text
τ → labeled as disorder / heresy / illegitimacy
→ O blocked
→ ε accumulates
→ Dᴀ increases
→ closure risk rises
```

## Function

Suppression preserves current authority in the short term but increases long-term misfit.

## Boundary

Not every innovation attempt is valid.

But every innovation conflict should be tested as a possible re-opening trigger before being sealed.
