# Operational Cost vs Restoration Cost

SΔϕ-27 separates two cost layers.

## Operational Cost

```text
C_op(P) = cost to continue along selected path
```

## Restoration Cost

```text
C_rest(P) = cost to abandon, revise, or reopen fixed path
```

## Core distinction

A path may be cheap in forward operation and expensive in reverse modification at the same time.

```text
low C_op(P)
can coexist with
high C_rest(P)
```

## Core warning

```text
Operational cheapness does not negate irreversibility.
It produces it.
```

## Boundary

Do not confuse smooth continuation with editability.

Do not confuse operational success with reversibility.
