## Core compression

```text

Operational cost =
cost required to continue along a selected path.

Restoration cost =
cost required to abandon, revise, or reopen a fixed path.

Exception compression =
reduction of cases requiring special handling, reinterpretation, or local repair.

Low-cost convergence =
a path is repeatedly selected because continuation is cheaper than alternatives.

Path fixation =
repeated selection structurally privileges a path and makes revision harder.
```
