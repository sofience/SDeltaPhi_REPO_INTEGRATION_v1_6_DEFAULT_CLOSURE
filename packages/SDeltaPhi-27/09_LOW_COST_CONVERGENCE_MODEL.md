# Low-Cost Convergence Model

Low-cost convergence occurs when a path is repeatedly selected because its continuation is cheaper than competing alternatives.

```text
C_op(P)↓ -> repeated selection↑
```

## Mechanism

```text
exception frequency decreases
-> operational cost decreases
-> path is selected again
-> repetition increases
```

## Result

Convenience becomes structure.

```text
practical convenience
-> repeated selection
-> structural asymmetry
```

## Boundary

Low-cost convergence is not proof of truth.

Low-cost convergence is not proof of universal adequacy.

It is a path-selection dynamic.
