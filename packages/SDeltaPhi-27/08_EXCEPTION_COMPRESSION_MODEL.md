# Exception Compression Model

Exception compression is the reduction of cases requiring special handling, reinterpretation, or local repair.

```text
E(P)↓ -> C_op(P)↓
```

## Mechanism

When a rule applies smoothly across many cases:

```text
special repair decreases
interpretation narrows
local variance is absorbed
routine continuation becomes cheaper
```

## Function

Exception compression makes a path feel natural.

## Boundary

The path does not feel natural because it is metaphysically privileged.

It feels natural because fewer exceptions demand intervention.
