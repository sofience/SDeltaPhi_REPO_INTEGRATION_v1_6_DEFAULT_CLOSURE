# Minimal Prompts

## Prompt 1 — Low-cost Convergence

Apply SΔϕ-27.
Classify whether path P is being repeatedly selected because C_op(P) is lower than alternatives.

## Prompt 2 — Cost Layer Split

Apply SΔϕ-27.
Separate C_op(P) from C_rest(P).

## Prompt 3 — Exception Compression

Apply SΔϕ-27.
Check whether E(P) is decreasing and lowering operational cost.

## Prompt 4 — Path Fixation

Apply SΔϕ-27.
Determine whether repeated selection R(P) is increasing fixation F(P).

## Prompt 5 — Restoration Cost Explosion

Apply SΔϕ-27 and SΔϕ-26.
Check whether F(P) is raising C_rest(P) toward or beyond the practical return threshold.

## Prompt 6 — Successful Routine Risk

Apply SΔϕ-27.
Evaluate whether a smoothly functioning routine is suppressing exception visibility.

## Prompt 7 — Authority Hardening

Apply SΔϕ-27 and SΔϕ-21.
Check whether low friction is being mistaken for truth, legitimacy, or universal adequacy.

## Prompt 8 — Editability Monitoring

Apply SΔϕ-27.
Monitor E(P), C_op(P), R(P), F(P), C_rest(P), and alternative route exercise frequency.

## Prompt 9 — Anti-overclaim

Apply SΔϕ-27.
Do not infer that efficiency is bad, all routines are dangerous, or successful systems are false.
