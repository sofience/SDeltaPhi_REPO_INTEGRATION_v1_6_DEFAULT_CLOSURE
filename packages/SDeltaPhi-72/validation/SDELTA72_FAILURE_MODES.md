# SDeltaPhi-72 Failure Modes

1. Efficiency Collapse - treating efficient structures as useful without cost audit.
2. Romantic Difficulty - treating costly or difficult structures as useful merely because they are hard.
3. Dislike Labeling - calling an unwanted institution useless without diagnosing closure.
4. Moralization Error - using useful/useless as praise/blame rather than transition analysis.
5. Default Attack Error - directly negating a thickened default on the same axis and increasing defensive cost.
6. Historical Demonization - assigning structural closure to a single person without accounting for institutional reproduction.
7. AI Replacement Overclaim - using AI creativity as proof that human creativity is obsolete rather than as evidence of a new judgment axis.
8. Re-entry Blindness - ignoring whether alternative paths can re-enter after the useful label is applied.
9. Habitation Blindness - ignoring how many actors live inside a default and therefore experience direct negation as world loss.
10. Closure Disguised as Usefulness - accepting practical, realistic, orthodox, or efficient labels as sufficient evidence of usefulness.
