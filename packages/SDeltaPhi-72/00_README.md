## Core Claim

Useful is not mere efficiency. Useless is not mere inefficiency.

SDeltaPhi-72 defines:

- **Useful** as transition-opening capacity.
- **Useless** as closure-producing efficiency.

A structure may be high-cost yet useful if it opens a new question, reveals hidden cost, preserves re-entry, or creates a new judgment axis. A structure may be efficient yet useless if it closes inquiry, hides cost, and thickens default power.

## Quick Routing

Use this package when evaluating claims about usefulness, efficiency, practicality, orthodoxy, realism, or uselessness.

Do not use it to label disliked theories, people, traditions, or institutions as useless without diagnosing closure, cost visibility, re-entry, and transition-opening capacity.
