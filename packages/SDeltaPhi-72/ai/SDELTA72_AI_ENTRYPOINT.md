# AI Entry Point - SDeltaPhi-72

Use this module when the user discusses:

- useful / useless
- efficiency / inefficiency
- practicality / realism
- orthodoxy / default power
- cost visibility
- question-opening value
- closure-producing labels
- new judgment axes
- re-entry paths
- AI creativity and human-only creativity defaults

## Core Diagnostic Question

Does the structure open transition, or does it close transition while appearing useful?

## Primary Rules

```text
Useful is not mere efficiency.
Useless is not mere inefficiency.
Useful = transition-opening capacity.
Useless = closure-producing efficiency.
```

## Ask

1. What question does this structure open?
2. What question does it close?
3. Whose cost does it reveal?
4. Whose cost does it hide?
5. Does it preserve re-entry?
6. Does it create a new judgment axis?
7. Does it call itself useful while producing closure?
