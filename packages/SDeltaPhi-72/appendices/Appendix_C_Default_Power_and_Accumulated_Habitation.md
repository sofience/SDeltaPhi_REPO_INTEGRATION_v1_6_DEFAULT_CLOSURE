# Appendix C - Default Power and Accumulated Habitation

A default power becomes thicker when many actors live inside it. Time does not automatically weaken a default. It can strengthen it through education, memory, institution, authority, identity, ritual, status, and emotional investment.

Directly attacking the default on the same axis may increase defensive cost. A transition-opening question often works by creating a different judgment axis that reveals what the default cannot see.

```text
Default power is not weakened merely by age; it is often thickened by accumulated habitation.
```
