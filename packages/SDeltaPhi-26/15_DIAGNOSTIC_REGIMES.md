# Diagnostic Regimes

## Low-cost Reversible Transition

```text
C_restore(t) < theta
return remains selectable
```

## Rising Restoration Cost

```text
C_restore(t) approaches theta
reversal still possible but increasingly costly
```

## Event Fixation

```text
C_restore(t) >= theta
low-cost return closed
event becomes fixed
```

## Formal-only Reversibility

```text
prior state can be described
but return path is not practically selectable
```

## Digital Pseudo-reversibility

```text
delete / rollback possible locally
but propagated traces remain
```

## Cost-exploded Irreversibility

```text
return formally conceivable
but restoration cost exceeds practical return threshold
```
