# Low-cost Return Closure

SΔϕ-26 defines event fixation through closure of low-cost return.

```text
low-cost return closes
→ event fixation
```

## Meaning

A transition may remain formally reversible while becoming practically irreversible.

Return is not impossible in principle.

Return is no longer selectable within the operative cost structure.

## Core thesis

```text
What fixes the world is not the impossibility of undoing,
but the closure of acceptable return.
```

## Boundary

Do not confuse possible description of the prior state with practical restoration of the prior state.
