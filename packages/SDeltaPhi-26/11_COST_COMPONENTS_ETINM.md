# Cost Components — E, T, I, N, M

SΔϕ-26 models restoration cost as:

```text
C_restore(t) = f(E, T, I, N, M)
```

## E — Energetic / Material Cost

The physical, material, or energetic burden of restoration.

## T — Temporal Cost

The time required to restore a prior state.

## I — Informational Reconstruction Cost

The burden of reconstructing the information needed to specify and restore the prior state.

## N — Network Propagation Cost

The burden created by copies, observers, downstream systems, distributed traces, and propagation.

## M — Memory / Interpretive Residue Cost

The burden created by memory, interpretation, reputational residues, and meaning already formed around the transition.

## theta — Practical Return Threshold

The cost threshold at which return ceases to be practically selectable.
