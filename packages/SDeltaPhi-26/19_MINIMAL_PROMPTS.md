# Minimal Prompts

## Prompt 1 — Irreversibility

Apply SΔϕ-26.
Classify irreversibility as restoration cost exceeding the threshold of practical return.

## Prompt 2 — Reversibility

Apply SΔϕ-26.
Check whether C_restore(t) is below theta.

## Prompt 3 — Event Fixation

Apply SΔϕ-26.
Determine whether Fixation(e) holds: C_restore(e) >= theta.

## Prompt 4 — Formal vs Practical Reversibility

Apply SΔϕ-26.
Separate describable prior state from practically selectable restoration path.

## Prompt 5 — Digital Pseudo-reversibility

Apply SΔϕ-26.
Check whether deletion or rollback restores the total prior world-state or only a local layer.

## Prompt 6 — Cost Components

Apply SΔϕ-26.
Estimate E, T, I, N, and M components of restoration cost.

## Prompt 7 — Eventhood

Apply SΔϕ-26 and SΔϕ-25.
Check whether eventhood = detected difference + irreversible fixation.

## Prompt 8 — Anti-overclaim

Apply SΔϕ-26.
Do not infer metaphysical finality, legal finality, moral finality, or absolute undo impossibility.
