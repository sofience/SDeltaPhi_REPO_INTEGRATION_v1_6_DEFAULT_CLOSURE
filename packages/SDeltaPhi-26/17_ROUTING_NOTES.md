# Routing Notes

SΔϕ-26 should be called when the task involves irreversibility, reversibility, restoration cost, event fixation, rollback, practical return, deletion, digital traces, virtual rollback, or historical consequence.

## Primary route

```text
SΔϕ-25 → SΔϕ-26
```

## Full route

```text
SΔϕ-03 → SΔϕ-25 → SΔϕ-26
```

## Eventhood route

```text
SΔϕ-24 → SΔϕ-25 → SΔϕ-26
```

## Call SΔϕ-26 when the task involves

- irreversibility;
- restoration cost;
- reversibility;
- rollback;
- event fixation;
- practical return threshold;
- formal vs practical reversibility;
- deletion vs restoration;
- digital pseudo-reversibility;
- propagated traces;
- broken cup / internet trace / virtual rollback examples.

## Do not call SΔϕ-26 merely because something changed

If the task asks about latent differences before detection, route through SΔϕ-25.

If the task asks about innovation or re-opening, route through SΔϕ-24.

If the task asks about legal finality, use domain-specific legal analysis.

If the task asks about moral consequence, use ethics analysis and do not treat SΔϕ-26 as a moral theory.
