# Eventhood as Detected Difference + Fixation

SΔϕ-26 defines full eventhood as:

```text
Eventhood = detected difference + irreversible fixation
```

## Meaning

Many transitions occur.

Only some become fixed events.

A transition becomes an event in the full SΔϕ sense when:

```text
difference passes from latent activity into registration
+
restoration cost rises beyond practical return threshold
```

## Boundary

Not all registered differences become fixed events.

Not all operative differences become historically consequential.

Eventhood requires both registration and fixation.
