# Event Fixation Model

Event fixation occurs when reversal is no longer selected within the available cost structure.

```text
Fixation(e) iff C_restore(e) >= theta
```

## Meaning

Something happening is not enough to make it a fixed event.

A transition becomes fixed when low-cost return closes.

## Sequence

```text
difference detected
→ restoration cost rises
→ low-cost return closes
→ event fixation
```

## Boundary

Occurrence alone is not full eventhood.

Event fixation requires practical non-return.
